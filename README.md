# react-native-polar-ble

React Native bridge for polar-ble-sdk

## Installation

```sh
npm install react-native-polar-ble
```

## Usage

```js
import {multiply} from 'react-native-polar-ble';

// ...

const result = await multiply(3, 7);
```
